## Decisions Closed This Session

### 1. Propulsion Architecture — Series Hybrid Confirmed

**Decision:** Series hybrid propulsion is confirmed as the MAOS propulsion architecture.

**Rationale:** The series hybrid approach uniquely satisfies the failure redundancy requirement. No single failure produces a total power loss. The ICE generators are mechanically isolated from the propulsor — they can be loaded, unloaded, or shut down independently without affecting thrust. This architecture is the only configuration that provides three independent failure modes between the crew and a total power loss event.

**Propulsion agent note:** The HV bus voltage remains an open gate. Candidates are 400V and 800V. The 800V bus is preferred from a conductor weight standpoint but requires more careful motor selection. Decision required before motor procurement.

**Safety agent note:** The failure matrix has been completed for the confirmed architecture. It is included in DWG MAOS-PM-002. No single credible failure produces a total power loss. This satisfies the safety case requirement.

### 2. Fuselage Configuration — Pod-and-Boom Locked

**Decision:** Pod-and-boom fuselage configuration is locked. Cylindrical pod (~52" W × 58" H × 156" L) on a tail boom. High wing above the pod with V-strut bracing.

**Rationale:** The pod-and-boom configuration provides a clean interface between the pressurized pod and the wing. No pneumatic systems cross the wing-to-pod boundary. The entire pressurization system lives within the pod. Wing attachment is purely structural and electrical. This dramatically simplifies the v1.0 to v1.1 pressurization upgrade path.

**Structures agent note:** The cylindrical pod cross-section is structurally efficient for pressurization loads. The exact pod wall thickness and material schedule is an open gate pending the pressurization design session.

### 3. Tail Configuration — Conventional Aft-Tail

**Decision:** Conventional aft-tail. Not T-tail. Not canard.

**Rationale:** The conventional aft-tail is the most buildable configuration for the target builder skill level. T-tail introduces flutter risk and complexity at the fin/horizontal stab junction. Canard configurations, while aerodynamically interesting, impose significant CG management complexity that conflicts with the variable payload mission (4 seats, variable occupancy, baggage).

**Aerodynamics agent note:** AVL stability analysis confirms positive static margin at both CG extremes with the conventional aft-tail configuration. Results logged in the stability analysis folder.
