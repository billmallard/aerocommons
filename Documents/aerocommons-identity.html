Primary palette

Deep Navy and Signal Orange are the two load-bearing colors. Every document, product, or print application should be traceable to this pair. Field Cream is the preferred background for print and physical surfaces.

Accent usage rule

Signal Orange should appear in small doses — the origin dot, a rule, a callout, a heading accent, a CTA button. It should never fill a large background area except in special-use applications. When in doubt, use less orange.

04
Clear Space & Minimum Size

Minimum size — print

Horizontal lockup minimum: 60mm wide. Mark only: 10mm wide. Below these sizes, use the single-line mark only.

Minimum size — screen

Horizontal lockup minimum: 200px wide. Mark only: 48px wide. At 32px and below, use the favicon variant (single line + dot only).

05
Usage Rules
Do
Use the logo on approved backgrounds: Deep Navy, Field Cream, white, and Signal Orange.
Maintain the three-line hierarchy — bold primary, medium secondary, thin tertiary. Never flatten all three to the same weight.
Use Signal Orange exclusively for the lead streamer line and origin dot. Optionally extend to a single accent element per layout (rule, CTA, tag).
Use Barlow Condensed for all display and technical typography. Use Barlow for body copy.
Reproduce the logo from the master SVG file. Never recreate from a raster screenshot.
Maintain clear space equal to the cap height of COMMONS on all four sides.
Don't
Don't rotate or skew the logo. The horizontal lockup is the primary form.
Don't apply drop shadows, glows, or gradients to the mark.
Don't place the logo on a busy photographic background without a containing field of brand color.
Don't recolor the origin dot. It must always be Signal Orange (#E8702A).
Don't separate AERO from COMMONS or use either word alone as the brand name in a logotype context.
Don't use the logo at sizes below the minimum — use the favicon mark variant instead.
